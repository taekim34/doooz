## Day 1: 登録してファミリーを作ろう

### まずは保護者が登録します

DOOOZは、保護者がファミリーを作り、子どもが招待コードで参加する仕組みです。最初の保護者がメールアドレスで登録するところから始まります。

1. アプリを開くとログイン画面が表示されます。**「新規登録」**をタップしてください。
2. メールアドレスとパスワード（6文字以上）を入力し、**「ファミリーを作る」**ボタンをタップします。
3. 認証メールが届きます。メールを開いてリンクをクリックすれば認証完了！
4. ログイン画面に戻り、メールアドレスとパスワードでログインします。

> **ヒント:** 認証メールが見つからない場合は、迷惑メールフォルダを確認してください。

---

### ファミリーを作ります

ログインすると、すぐにファミリー作成画面に移動します。

1. **ファミリー名**を決めます。（例：「田中ファミリー」「ハッピーハウス」）
2. **自分の名前**を入力します。（例：「ママ」「パパ」「太郎ママ」）
3. **招待コード**を自分で決めるか、空欄にすると自動生成されます。（例：「うちの家族」）
4. **タイムゾーン**を選択します。日本ならAsia/Tokyoを選んでください。
5. **言語**を選択します。한국어、English、日本語から選べます。
6. **「ファミリーを作る」**ボタンをタップすると、キャラクター選択画面に進みます。
7. 好きなキャラクターを1つ選んでください。12種類から選べます！

> **補足:** ファミリー名はログインIDの役割を果たすため、他のファミリーと重複できません。招待コードはいつでもファミリータブ → 招待で確認できます。

---

### 子どもや他の家族を招待します

ファミリーができたら、子どもや他の保護者を招待しましょう。子どもはメールアドレスなしで登録できます！

1. 招待する人のデバイスでアプリを開きます。
2. ログイン画面で**「招待コードで参加（メール不要）」**リンクをタップします。
3. **ファミリー名**を入力します。（ファミリーを作った保護者に聞いてください）
4. **招待コード**を入力します。
5. **自分の名前**を決めます。（例：「太郎」「花子」）
6. **パスワード**（6文字以上）を設定します。
7. **役割**を選択します — 「子ども」または「保護者」。役割によって使える機能が異なります。保護者はタスクの登録・管理、大目に見る、ごほうび管理などができます。子どもはタスクのチェック、おねだり、ごほうび申請、キャラクター変更などができます。
8. **「ファミリーに参加」**ボタンをタップし、キャラクターを選んだら完了！

> **ログイン方法:** すべての家族メンバーは「ファミリー名で」タブから**ファミリー名 + 名前 + パスワード**でログインします。最初の保護者は「メールで」タブからメールアドレス + パスワードでもログインできます。

## Day 1: 保護者がタスクを登録します

子どもたちが参加したら、さっそくタスクを作ってみましょう！

1. 下部ナビゲーションの**「タスク」**タブをタップします。
2. 右上の**「管理」**リンクをタップします。
3. **「タスクを追加」**ボタンをタップします。

### くり返しタスクを登録する（毎週くり返すもの）

4. **タイトル**を入力します。（例：「本を読む」「部屋を片付ける」）
5. **担当する子ども**を選びます。
6. **ポイント**を決めます — かんたん（50pt）、ふつう（100pt）、むずかしい（150pt）から選ぶか、「自分で入力」で好きなポイントを設定できます。
7. **くり返す曜日**を選びます。（例：月〜金だけチェック、または毎日）
8. **開始日**を決めます。
9. 必要であれば**終了日**も設定します。（任意）
10. **「保存」**をタップすれば登録完了！

### 1回きりの目標を登録する

4. タイトルを入力します。（例：「読書感想文を書く」）
5. 担当する子どもを選びます。
6. ポイントを決めます。
7. くり返しを**「1回」**に設定します。
8. **期限日**を決めます。（この日までに達成すればOK）
9. **「保存」**をタップすれば登録完了！

> **💡 ヒント:** 今日のタスクを追加することもできますが、大きなポイントを設定して長期的なチャレンジ目標にすることもできます。たとえば「AIバイブコーディングでアプリを作る（1000pt）」のように設定すれば、達成時に🎯 **大きな目標達成**バッジも一緒にゲットできます！

> **編集・削除:** 管理画面で既存タスクのタイトル、ポイント、担当者、開始日、終了日をすぐに編集できます。くり返し設定を変更するには、削除してから新しく追加してください。
> 
> **終了したタスク:** 終了日を過ぎたタスクは「終了済み」セクションに表示されます。もう必要なければ「完全に削除」で永久に削除できます。

## Day 2: 子どもがタスクをチェックします

### 子どものホーム画面

子どもがログインすると、ホーム画面にはこんなものが表示されます：

- 自分の**キャラクター**と**レベル進捗バー**（現在のポイント / 次のレベルまでに必要なポイント）
- 右側に**現在の保有ポイント**
- **今日のタスク**リスト
- これまでに獲得した**バッジ**コレクション

### タスクをチェックする

1. 下部ナビゲーションの**「タスク」**タブをタップします。
2. 今日のタスクリストが表示されます。完了したタスクを**チェック**しましょう！
3. チェックした瞬間にポイントが即座に加算されます！（「+100pt!」のようなメッセージが表示され、残高も確認できます）

> **チェック取り消しに注意:** 間違えてチェックした場合はもう一度タップして取り消せますが、ポイントが回収され、その間に獲得したバッジが取り消される可能性もあります！

### 予定されたタスク

今日以降に期限がある1回きりの目標がある場合、「予定されたタスク」セクションに**D-3**、**D-7**のようなカウントダウンとともに表示されます。事前に確認して準備しましょう。

### 見逃したタスク

昨日までに完了できなかったタスクは、赤い「見逃したタスク」セクションに表示されます。（深夜に自動で-50ptが差し引かれた状態です。）

### もっとやりたい？おねだり！

タスクリストにないことをやったのにポイントがほしい？「おねだり」機能を使ってみましょう。

1. タスクタブの下の方に**「おねだり」**セクションがあります。
2. やったことを書きます。（例：「お皿洗いを手伝ったよ」）
3. **「リクエスト」**ボタンをタップすると、保護者に通知が届きます。
4. 保護者が承認すればポイントがもらえます！

> **おねだりの状態:** リクエスト後は「保留中」リストに表示されます。気が変わったら自分でキャンセルもできます。保護者が却下する場合もあります。

## Day 2: 保護者が確認します

### 保護者のホーム画面

保護者がログインすると、ホーム画面に子どもたちの状況がカード形式で表示されます：

- 各子どもの**キャラクター**、**レベル**、**保有ポイント**
- 今日のタスク**進捗**（完了数 / 全体数 + 進捗バー）
- 各タスクの完了状況

> **詳細表示:** 子どものカードをタップすると、その子の詳細ページに移動します。統計、今日のタスク、バッジ、最近のポイント履歴が一覧で確認できます。

### おねだりリクエストを確認する

子どもがおねだりを送ると、プッシュ通知が届き、タスクタブにも黄色い「おねだりリクエスト」セクションが表示されます。

1. タスクタブで**「おねだりリクエスト」**を確認します。
2. 子どもの名前とリクエスト内容を見て、**「承認」**または**「却下」**をタップします。
3. 承認する際は、ポイントを自分で決めて付与できます！

### 大目に見る

子どもがタスクをできなかったけど事情があった？「大目に見る」でペナルティを免除できます。

1. タスクタブで、子どもの未完了または遅れたタスクの横に**「大目に見る」**ボタンがあります。
2. タップすると、そのタスクが「免除」ステータスに変わります。
3. すでに-50ptのペナルティが差し引かれていた場合、自動的に+50ptの補填が入ります。
4. まだ差し引かれていない場合は、今後の差し引きを防ぎます。

> **大目に見るの取り消し:** 間違えて大目に見た場合は、取り消すこともできます。

## Day 3: 夜中に何が起こる？

DOOOZは毎日自動的に2つのことを行います：

### 午前1時（自動処理）

- 昨日完了できなかったタスクは**「遅刻」**ステータスに変わり、1件あたり**-50pt**が差し引かれます。
- 今日の日付に該当するくり返しタスクが**自動的に新規生成**されます。

### 午後9時（リマインダー）

- まだ完了していないタスクがある子どもに**プッシュ通知**が届きます。
- 「今日のタスクが残っています！残り3件：本を読む、部屋を片付ける、日記を書く」のようなメッセージです。

> **ヒント:** 午前1時までにチェックすればペナルティを回避できます！午後9時のリマインダーをしっかり確認しましょう。

## Day 4〜5: ポイントが貯まったらごほうび！

### 保護者：ごほうびを登録する

子どもたちがタスクをがんばればポイントが貯まりますよね？ポイントと交換できるごほうびを作ってあげましょう。

1. 下部ナビゲーションの**「ごほうび」**タブをタップします。（保護者は自動的にごほうび管理画面に移動します）
2. **「新しいごほうび」**でごほうびの名前を入力します。（例：「アイスクリーム」「映画を観る」）
3. **必要ポイント**を決めます。（例：500）
4. **「追加」**をタップすれば登録完了！

> **削除:** もう提供しないごほうびは削除できます。

### 子ども：ごほうびを申請する

1. 下部ナビゲーションの**「ごほうび」**タブをタップします。
2. 登録されたごほうびのリストと、それぞれの必要ポイントが表示されます。
3. 右上に現在の保有ポイントも表示されます。
4. ほしいごほうびの**「ほしい！」**ボタンをタップします。
5. 保護者が承認するとポイントが自動で差し引かれます。実際のごほうび（おこづかい、特別なイベントなど）はオフラインで渡すことを前提としており、アプリ内で受け渡しの確認はしません。

> **申請のキャンセル:** 気が変わったら、保留中の申請をキャンセルできます。
> 
> **却下されたら？** 保護者が却下した場合は理由が表示されます。ポイントは差し引かれません。
> 
> **申請状況:** ごほうびタブの下部で、保留中、承認済み、却下済み、キャンセル済みのすべての状態を確認できます。

## Day 7: レベルアップとバッジ！

### レベルシステム（30段階）

ポイントが貯まると自動的にレベルが上がります！タスクを完了するたびに「レベルアップ！」メッセージが表示されることがあります。

- Lv.1 わかば → Lv.5 大樹 → Lv.10 流星 → Lv.15 永遠 → Lv.20 不滅 → Lv.25 永劫 → Lv.30 超越
- 全30段階、累計ポイントに基づいて自動レベルアップ！

### バッジシステム（58種）

さまざまな条件を達成すると自動的にバッジがもらえます。ホーム画面とキャラクターページで獲得したバッジを確認できます。

> **バッジは取り消される場合があります:** タスクのチェックを取り消すと条件を満たさなくなり、バッジが消えることがあります。慎重に！

### キャラクター進化（5段階）

選んだキャラクターはレベルに応じて5段階で進化します：

- **ひよこ** → **ルーキー** → **戦士** → **英雄** → **伝説**

### キャラクターの変更

1. 下部ナビゲーションの**「キャラクター」**タブをタップします。
2. **「キャラクターギャラリー」**でロック解除されたキャラクターをタップすれば変更できます。

> **補足:** キャラクター変更は子どもだけがキャラクタータブから直接行えます。保護者は設定からキャラクター変更リンクに移動します。

## その他の便利な機能

### 過去のタスクを振り返る

過去に何をしたか確認したい場合は？

1. **「タスク」**タブで**「履歴」**リンクをタップします。
2. 「← 前の日」/「次の日 →」ボタンで日付を移動して確認できます。
3. 保護者は子どもごとにフィルタリングし、過去のタスクの完了状態を変更したり大目に見ることができます。
4. 子どもは読み取り専用で履歴のみ確認できます。

> **保護者向けヒント:** 「子どもが間違えてチェックした場合や修正が必要なタスクは、ここで取り消し・再完了できます。」

### ポイントを確認する

1. 下部ナビゲーションの**「ポイント」**タブをタップします。
2. **現在の残高**と**総獲得ポイント**が一目で確認できます。
3. 保護者は子どもごとの残高と最近の取引履歴も表示されます。
4. **「ポイント履歴」**で、すべての加算・差し引き履歴を時系列で確認できます。（50件ずつ「もっと見る」で読み込み）

### ファミリーを見てみよう

1. 下部ナビゲーションの**「ファミリー」**タブをタップします。
2. すべての家族メンバーのリストが表示されます。
3. メンバーをタップすると詳細プロフィールページに移動します：
   - キャラクター、レベル、段階の進捗
   - 活動日、登録日などの統計
   - 今日のタスク状況
   - 獲得したバッジ
   - 最近のポイント履歴

> **招待コードの確認:** ファミリータブ → 「招待」で招待コードを確認・コピーできます。新しい家族メンバーを追加する際に使ってください。

### 設定を変更する

下部ナビゲーションの**「設定」**タブをタップすると：

- **名前の変更** — 表示名を変更できます。
- **パスワードの変更** — 現在のパスワードを入力して新しいパスワード（6文字以上）に変更。
- **キャラクターの変更** — ギャラリー画面に移動。
- **ファミリー設定**（保護者のみ） — ファミリー名、言語（한국어/English/日本語）、タイムゾーンの変更。
- **ログアウト**

### アプリのようにインストール（PWA）

DOOOZはインストールなしでブラウザからそのまま使えますが、**ホーム画面に追加すると、完全な独立したアプリとしてすべての機能を使えます！**

**Android:**

1. ChromeでDOOOZにアクセスします。
2. 右上のメニュー（点3つ）→ **「ホーム画面に追加」** → **「アプリをインストール」**の順にタップします。

**iPhone:**

1. SafariでDOOOZにアクセスします。（必ずSafariで！）
2. 下部の共有ボタン（四角に矢印）→ **「ホーム画面に追加」**をタップします。「Webアプリとして開く」がオンになっていることを確認して**「追加」**をタップすれば完了！

> **通知を有効にする:** インストール後、通知の権限を許可してください。おねだり、ごほうび、リマインダーなどのプッシュ通知を受け取れるようになります。

### プッシュ通知の種類

DOOOZから届くプッシュ通知はこちらです：

| 状況            | 誰に届く？ | 内容                               |
| ------------- | ----- | -------------------------------- |
| おねだりリクエスト     | 保護者   | 「太郎が『お皿洗いを手伝ったよ』を認めてほしいと言っています！」 |
| おねだり承認        | 子ども   | 「『お皿洗いを手伝ったよ』が承認されました！+100pt」    |
| おねだり却下        | 子ども   | 「『お皿洗いを手伝ったよ』のリクエストが却下されました」     |
| おねだりキャンセル     | 保護者   | 「太郎がリクエストをキャンセルしました」             |
| ごほうび承認        | 子ども   | 「『アイスクリーム』のごほうびが承認されました！」        |
| ごほうび却下        | 子ども   | 「『アイスクリーム』のごほうびが却下されました」         |
| 夜のリマインダー（21時） | 子ども   | 「今日のタスクが残っています！残り3件：…」           |

> **補足:** タスクのチェック・取り消しは頻度が高く通知が多くなりすぎるため、プッシュ通知の対象外としています。
